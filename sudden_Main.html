프로젝트 기획의도

기능 설명

1. 회원관리

- 회원제를 위한 테이블 생성 및 회원제 기능 구현

- 회원가입, 로그인 및 마이 페이지 기능 구현

- 로그인 시 가상 session(변수)에 저장하여 유동적으로 설계됨. 로그아웃 시 변수 내 값을 삭제함

2. 마이 페이지 관리

- 마이 페이지 내에서 회원정보 수정 및 탈퇴 기능 구현

- 마이 페이지 내에서 카카오페이 API를 이용 한 캐쉬 충전 기능 구현

3. 커뮤니티 관리

- 회원들이 이용 할 수 있는 자유 게시판 기능

- admin 계정으로 작성 할 수 있는 공지사항 게시판 기능

- select 박스로 작성자/글 제목으로 검색 할 수 있는 검색창 기능

- 글 상세 조회 및 ajax로 댓글 기능 구현

4. 게시글 관리

- 자유 게시판 , 신호 게시판 , 공지 게시판 글 작성

- 신호 게시판 선택 시 나의 신호등 목록에 있는 신호를 하나 선택 후 같이 등록함

- 신호 게시판에 등록된 신호등을 나의 신호등 목록에 저장하기

5. 거래소

- 회원들이 아이템을 사고 팔 수 있는 거래소 기능

- 한 페이지에 아이템이 9개씩 출력 되며 그 이상 넘어가면 다음페이지로 이동되는 페이징 기능

- 아이템을 구매 하면 그 창에서 사라지며 내가 구매한 목록에 추가되는 기능

- 아이템을 판매 및 구매 할 시 거래내역에 자동으로 추가 되는 거래내역 기능

6.고객센터

- 실시간 운영자와 채팅 할 수 있는 채팅기능 구현

7. 관리자

- 회원 목록 조회 및 강제탈퇴 기능

- 불필요하다고 생각되는 게시물 강제 삭제 기능

- 도배성으로 의심되는 아이템 삭제 기능
