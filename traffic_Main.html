나의 역할

1. 회원관리

- 회원제를 위한 테이블 생성 및 회원제 기능 구현

- 회원가입, 로그인 및 마이 페이지 기능 구현

- 로그인 시 가상 session(변수)에 저장하여 유동적으로 설계됨. 로그아웃 시 변수 내 값을 삭제함

2. 마이 페이지 관리

- 마이 페이지 내에서 회원정보 수정 및 탈퇴 기능 구현

- 내가 쓴 글 보러 가기 및 마이 플레이스, 내가 등록한 신호등 보기 가능

3. 게시판 관리

- 자유 게시판 , 신호 게시판 , 공지사항 게시판 제작

- 검색창에 검색 시 해당 단어가 들어간 게시글 출력

- select 박스로 시/도 , 군/구 선택 시 해당 지역의 게시글 출력

- 글 상세 조회 및 글 추천 비추천 기능

4. 게시글 관리

- 자유 게시판 , 신호 게시판 , 공지 게시판 글 작성

- 신호 게시판 선택 시 나의 신호등 목록에 있는 신호를 하나 선택 후 같이 등록함

- 신호 게시판에 등록된 신호등을 나의 신호등 목록에 저장하기

5. 신호등 및 게시글 신고

- 게시글 신고 기능, 신고 시 관리자의 게시글 신고 목록에 저장됨

- 신호등 신고 기능, 신고 시 관리자의 신호등 신고 목록에 저장됨

6. 데이터 베이스 설계 및 쿼리문 작성

-첫 설계 ERD 제작 , 프로젝트 도중 추가 ,변경되는 사항에 대해서 데이터베이스 수정 처리

-Spring Boot Jpa 의 기본 키가 아닌 쿼리 문들을 팀원들에게 요청 받고 처리

7. 관리자

- 회원 등급 조회 및 강제탈퇴 기능

- 신고 받은 게시글 출력 , 조치 후 내용 입력 팝업 창 띄우기

- 관리자 신호등 리스트 출력 및 관리자 전용 신호등 등 페이지 생성

- 신호 게시판에서 일정 추천 수를 받은 글을 관리자 추천 신호에 띄우고 해당 신호의 정보를 볼 수 있음

- 관리자가 처리한 내용들을 볼 수 있는 관리자 history 리스트 출력

기타 작업

-팀원들이 작성한 코드들을 하나로 합치고 그에 대한 에러코드 및 오류를 검출해 수정함

-팀원들의 작업 현황을 매일 조사하고 현황에 따라 스케줄이나 사양을 변경함

-모든 프로젝트 관련 데이터베이스와 서류를 보관하고 관리함
